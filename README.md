# Alphi's Training

A simple personal tracker for the 10 recommended Bark Busters home-training exercises.
Built as a single self-contained `index.html` — opens locally or via GitHub Pages.

Five views:
- **Today** — tap-to-log the day's exercises with optional per-exercise notes
- **Schedule** — 5×7 grid mirroring the Bark Busters tracking sheet
- **How To** — verbatim exercise instructions, formatted to scan on mobile
- **Journal** — quick free-text per day, listed newest-first
- **Settings** — set the program start date, export / import / reset

Data lives in `localStorage` on the device that opens the page.
Use Settings → Export JSON to back up.
